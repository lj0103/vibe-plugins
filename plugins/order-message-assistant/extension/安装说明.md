# 飞鸽真实输入配套扩展

1. 在 Chrome 地址栏打开 `chrome://extensions`。
2. 打开右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本插件目录下的 `extension` 文件夹。
5. 确认扩展“订单消息助手·飞鸽真实输入”处于开启状态。
6. 刷新快递助手和飞鸽页面。

扩展使用 Chrome `debugger` 权限，仅允许在 `im.jinritemai.com/pc_seller_v2/main/workspace` 页面输入订单消息助手本次传入的消息（最多 800 字），并实时定位、点击飞鸽的发送按钮。当前蓝旗消息只需在 Tampermonkey 主脚本的 `BLUE_FLAG_MESSAGE` 中修改。运行时 Chrome 可能短暂显示“正在调试此浏览器”。
